# SaveSync Vision

## Where We Are

SaveSync is a desktop application that protects game saves through automatic backup to local and cloud destinations. It has been engineered with a strong emphasis on reliability, recoverability, and safe failure handling.

## Today

Making SaveSync the most trustworthy game save protector available. Priorities are driven by user feedback, not a calendar. Areas we're actively exploring include:

- Game auto-detection — scanning known save locations so setup takes seconds
- Easier setup — reducing friction from download to first backup
- Automatic updates — no manual downloads for fixes
- Better background operation — system tray presence, notifications

## Next

Once the app has proven itself with real users, a Pro edition will add capabilities for users who need more — advanced scheduling, multiple profiles, priority support. The free edition remains genuinely useful for everyone.

## Future

Cloud services are only built when users actively request features that require a server — multi-device profile sync, restore version history, remote monitoring. These will be optional and never required for the desktop app to function.

## What Will Stay the Same

- **Local-first.** The app always works offline with local destinations.
- **Privacy-respecting.** No telemetry, no data collection without consent.
- **Reliability-focused.** Every feature proves it handles failure before it ships.
- **Windows-primary.** Other platforms follow only after Windows is excellent.

## What Will Never Drive Decisions

- Arbitrary release dates. Features ship when they're ready.
- Revenue growth at the expense of user trust.
- Feature count as a success metric.
- "Because competitors have it" as a reason to build.

## Success Isn't "Finished"

Success is:

- Users who trust SaveSync with their saves
- A reliable restore when someone needs it
- A sustainable product that improves over time
- Users willing to pay for Pro because they already rely on the free version

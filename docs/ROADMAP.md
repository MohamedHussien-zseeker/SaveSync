# SaveSync Roadmap

Intentional by design. This is not a commitment — it communicates direction.

---

## Current Phase

External beta testing (v2.4.0-beta1)

## Current Goal

Collect usability and reliability feedback from real users. Fix P0/P1 issues. Validate cloud OAuth end-to-end.

## After Beta

- Promote to v2.4.0 stable
- Game auto-detection — scan known save locations for one-click setup
- Installer — proper Windows installer (NSIS or Inno Setup)
- Automatic updates — no manual downloads for fixes

## Future Candidates

- System tray background mode
- Profile import/export
- Pro edition (advanced scheduling, multiple profiles, priority support)
- Multi-device sync via cloud storage

## Not Planned

- macOS or Linux as primary targets (Windows-first)
- Enterprise features (team management, central deployment)
- SaaS requirement for core functionality (local-first always)
- Telemetry or analytics of any kind
- Bundling third-party software or ads
